# FEKSS
本实验用于测试是否在一段面部视频中混入某个表情！本实验在一段语音视频中的中间片段（20%-80%）帧中混入了某一个表情，请在https://www.wjx.cn/vm/PWtcZ1s.aspx# 中对应的题目中选择对应的表情。本实验的结果保密，且仅用于该实验。
